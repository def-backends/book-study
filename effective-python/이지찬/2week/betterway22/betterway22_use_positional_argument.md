## 변수 위치 인자르르 사용해 시각적인 잡음을 줄여라
```bash
함수에서 어떤 데이터가 들어올지 모르는 경우 * 연산자를 통해 위치 기반 인자를 사용한다.
*연산자를 사용을 통해 받게되는 데이터의 타입은 튜플이다.
제너레이터의 경우 지연평가를 통해 필요할때마다 하나씩 데이터를 꺼내서 사용 되는데 
이때 * 연산자를 사용하게 되면 모든 데이터를 메모리에 올리기 때문에 OutOfMemory 오류가 발생할 수가 있다.
- 꼭 제너레이터가 아니더라도 데이터양이 크다면 어떤 거든 문제가 되지 않을까?
*연산자의 위치 인자 자리에 다른 위치 인자가 들어가게 된다면 예외가 발생하지 않고 코드가 작동 될 수 있기 때문에
감지 하기 힘든 버그가 발생 할 수 있다. 
```
