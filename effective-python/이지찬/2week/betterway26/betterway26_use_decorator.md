## functools.wrap을 사용해 함수 데코레이터를 정의하라
```bash
데코레이터를 사용하면 기존의 함수의 동작을 확장하거나 수정된 동작을 추가 할 수 있다.
데코레이터가 명시된 함수의 경우 함수가 호출이 되면 데코레이터가 실행이 되며
데코레이터에서는 인자값으로 호출된 함수를 받아서 wrapper 함수에서 호출된 함수를 실행 하게 된다.
이때, 이 wrapper 함수 내에서 호출된 함수의 동작을 추가 할 수 있고
결과값을 반환 한다.

@wraps 사용 했을때와 안했을때의 차이점?
- @wraps를 사용 했을때 데코레이터를 사용한 함수의 메타데이터(함수명, 주석, 인자값)등을 유지 할 수 있다.

표준 애트리뷰트
- __name__ : 함수의 이름
- __doc__ : 함수의 주석
- __module__ : 함수가 정의된 모듈의 이름
- __annotations__ : 함수의 주석을 통해 정의된 변수의 타입
- __defaults__ : 함수의 기본 인자값
- __kwdefaults__ : 함수의 키워드 인자값
- __dict__ : 함수의 사용자 정의 속성
- __wrapped__ : 래핑된 함수
등등
```
